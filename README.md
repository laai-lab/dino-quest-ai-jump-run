# Dino Quest: AI Jump Run

A browser-based 2D educational dinosaur jumping game built with plain HTML, CSS, and vanilla JavaScript using the Canvas API.

## Play

Open `index.html` in a browser.

## Replit

Import this GitHub repository into Replit as a static HTML/CSS/JS project. No backend, framework, package install, or build step is required.

## Controls

- Run: `A` / `D` or arrow keys
- Double jump: press `W`, `ArrowUp`, or `Space` once, then press again in the air
- Duck: `S` or `ArrowDown`

## Gameplay

Run left to right, double-jump hazards, collect boosts, and solve DS, AI, ML, Coding, and GenAI puzzle gates. Boss gates require multiple correct answers.
